    How to Use the To-Do List

    Tips for using the To-Do List (1)

    1. Add a Task

    Enter your task in the input field at the top of the page. Click the "Add" button to add the task to your list.

    2. Edit a Task

    If you want to edit a task, click the "Edit" button next to the task. A prompt will appear where you can update the task. Enter the new task description and click "OK" to save your changes.

    3. Delete a Task

    To remove a task, click the "Delete" button next to it. The task will be removed from the list permanently.

    4. Persist Your Tasks

    Your tasks are automatically saved, even if you close the browser or refresh the page. The tasks will remain in the list until you delete them.
